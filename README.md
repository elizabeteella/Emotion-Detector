# Final Project

## Emotion Detector

This project is an AI-based web application that detects emotions from text using IBM Watson NLP.

## Description
The Emotion Detector analyzes a given text input and identifies the emotions expressed within it. The application uses a pre-trained Watson NLP model to return emotion scores such as joy, anger, sadness, fear, and disgust.

## Features
- Emotion analysis from user input text
- Integration with Watson NLP API
- Flask-based web deployment
- Error handling for invalid input
- Unit testing for application validation

## Technologies Used
- Python
- Flask
- Requests library
- IBM Watson NLP
